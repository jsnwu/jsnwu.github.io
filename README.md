# jsnwu.github.io

[Live](https://jsnwu.github.io/)

Clean, no-build GitHub Pages site inspired by the Chirpy layout.

## Structure

- `index.html`: home + latest posts
- `posts/`: individual post pages + `posts/index.html`
- `posts/posts.json`: post index (edit this to add posts)
- `assets/`: CSS/JS

## Local preview

Just open `index.html` in a browser.
